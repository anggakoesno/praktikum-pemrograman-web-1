1. Paragraf dan Line Break

Ini adalah contoh paragraf. Kita bisa membuat paragraf dengan menggunakan tag < p > di awal dan < /p > di akhir. Isi atau tulisan dari paragraf bisa kita letakkan di antara kedua tag tersebut. Tag < p > disebut sebagai tag pembuka, sedangkan tag < /p > disebut dengan tag penutup

Dalam menulis paragraf, tentu memerlukan sebuah tata letak untuk memisahkan antara satu paragraf dengan paragraf yang lain. Salah satu cara untuk melakukannya adalah dengan menggunakan line break. Di dalam HTML, line break dapat dibuat dengan menggunakan tag < br > (tanpa tag penutup). Berikut contohnya

5. Teks yang di-preformat

Teks yang di-preformat dapat dibuat dengan cara menggunakan tag pembuka < pre > dan tag penutup < /pre >. Tujuannya adalah untuk menampilkan teks sesuai dengan format atau tata letak yang berada di dalam tag tersebut. Berikut contoh penggunaanya:

            Ini     adalah teks
            yang di-preformat.
        

            for i = 1 to 10
                print(i)
            next i

Quotation panjang dapat dibuat dengan tag pembuka < blockquote > dan tag penutup < /blockquote >. Quotation pendek dapat dibuat dengan tag pembuka < q > dan tag penutup < /q >. Quotation dibuat dengan tujuan menandai kutipan dari sumber lain secara struktural dan semantik.




7. List Unordered dan Ordered

Berikut contoh list unordered:

  • List 1
  • List 2
  • List 3

Berikut contoh list ordered:

  1. List 1
  2. List 2
  3. List 3

List unordered dapat dibuat dengan tag pembuka < ul > dan tag penutup < /ul >, sedangkan list ordered dapat dibuat dengan tag pembuka < ol > dan tag penutup < /ol >. List ordered dibuat untuk menampilkan daftar terurut, sedangkan list unordered dibuat untuk menampilkan daftar tidak terurut.




8. Jenis-jenis List Ordered

Contoh list ordered bernomor:

  1. List 1
  2. List 2
  3. List 3

Contoh list ordered dengan huruf:

  1. List 1
  2. List 2
  3. List 3

Contoh list ordered dengan huruf kecil:

  1. List 1
  2. List 2
  3. List 3

Contoh list ordered dengan angka romawi:

  1. List 1
  2. List 2
  3. List 3

Contoh list ordered dengan angka romawi kecil:

  1. List 1
  2. List 2
  3. List 3

List ordered dengan angka dapat dibuat dengan tag pembuka < ol > dan tag penutup < /ol >. Sedangkan, list ordered dengan huruf dapat dibuat dengan cara menambahkan atribut "type" di dalam tag pembuka < ol >. Pengguna dapat mengisi nilai dari "type" sesuai dengan huruf yang akan dijadikan sebagai poin pertama dalam daftar urutan.




9. Jenis-jenis List Unordered

List dengan bullet berbentuk disc:

  • List 1
  • List 2
  • List 3

List dengan bullet berbentuk lingkaran:

  • List 1
  • List 2
  • List 3

List dengan bullet berbentuk kotak:

  • List 1
  • List 2
  • List 3

List unordered dapat dibuat dengan tag pembuka < ul > dan tag penutup < /ul >. Sama seperti list ordered, pengguna dapat mengisi nilai dari "type" sesuai dengan bentuk atau simbol yang akan dijadikan sebagai penanda poin dalam daftar.




10. List Bersarang dan List Definisi

Berikut ini contoh list bersarang

  • List 1
  • List 2
    • List 2.1
    • List 2.2
  • List 3

Berikut ini contoh list definisi

Definisi 1
Deskripsi 1
Definisi 2
Deskripsi 2

List bersarang adalah list yang menampung list lain di dalamnya. List bersarang dapat dibuat dengan membuat tag list seperti ul atau ol di dalam ul atau ol.

List definisi adalah list yang berisi pasangan antara poin dan deskripsi. List definisi dapat dibuat dengan tag pembuka < dl > dan tag penutup < /dl >.

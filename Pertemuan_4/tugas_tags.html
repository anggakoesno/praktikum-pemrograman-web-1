Membuat Contoh Elemen HTML Form

Tag < fieldset > dan < legend >

Data Diri

Fieldset berfungsi sebagai pembungkus atau kontainer elemen-elemen input. Legend berfungsi menentukan judul atau keterangan untuk fieldset


Tag < datalist >

Elemen datalist akan memberikan auto-complete dari data yang sudah disiapkan. Terlebih dahulu, mahasiswa membuat elemen input yang memiliki atribut list. Kemudian, mahasiswa dapat membuat elemen datalist dengan atribut id. Nilai dari atribut list milik elemen input harus sama dengan nilai dari atribut id milik elemen datalist. Di dalam datalist, mahasiswa dapat mengisi dengan elemen option.


Tag < optgroup > dan < option >

Optgroup berfungsi untuk mengelompokkan option dengan label tertentu.
